# CHƯƠNG I - PHẢN ỨNG HÓA HỌC | KHOA HỌC TỰ NHIÊN 8

Trang web học tập tương tác về Chương 1 - Phản ứng hóa học, sách giáo khoa Khoa học tự nhiên lớp 8.

## Nội dung

- **6 bài học** (Bài 2-7) với nội dung đầy đủ
- **17+ hình ảnh thực tế** (nhúng base64, không cần thư mục ảnh)
- **12+ hoạt động tương tác** (máy tính mol, mô phỏng phản ứng, quiz, v.v.)
- **12 câu hỏi trắc nghiệm** tổng hợp
- Font Arial, tiêu đề viết hoa, định dạng thống nhất

## Cách deploy lên Vercel

### Bước 1: Tạo repository trên GitHub
1. Đăng nhập vào [GitHub](https://github.com)
2. Click **New repository**
3. Đặt tên: `khtn-8-chuong-1-phan-ung-hoa-hoc`
4. Chọn **Public**
5. Click **Create repository**

### Bước 2: Upload file
1. Click **uploading an existing file**
2. Kéo thả file `index.html` và `vercel.json` vào
3. Click **Commit changes**

### Bước 3: Deploy trên Vercel
1. Đăng nhập vào [Vercel](https://vercel.com)
2. Click **New Project**
3. Chọn repository `khtn-8-chuong-1-phan-ung-hoa-hoc`
4. Framework Preset: **Other**
5. Click **Deploy**
6. Đợi 1-2 phút, trang web sẽ online!

## Cập nhật nội dung

Khi muốn sửa nội dung:
1. Sửa file `index.html` trên GitHub
2. Vercel sẽ tự động deploy lại
